1. Перейдіть до веб-сайту Flickr та увійдіть до вашого облікового запису
  2. Перейдіть до сторінки ключів API
  3. Натисніть Get another key
  4. Виберіть Apply for a non-commercial key
  5. Введіть ім'я та опис, і відмітьте обидва прапорці перш ніж підтвердити
  6. Тепер скопіюйте Key та Secret в налаштування Flickr2Piwigo
  7. Натисніть Edit auth flow for this app
  8. Введіть опис знову, в наступне поле вставте Callback URL та натисніть Save changes
  9. Ви це зробили!

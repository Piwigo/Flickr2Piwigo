1. Allez sur le site Flickr et connectez vous à votre compte
  2. Rendez-vous à la page des clefs API
  3. Cliquez sur Obtenir une autre clé
  4. Choisissez Demander une clé non commerciale
  5. Entrez un nom et une description et cochez les cases avant de valider
  6. Maintanant copiez la Clé et le Secret dans la configuration de Flickr2Piwigo
  7. Cliquez sur Modifier le flux d'authentification
  8. Entrer à nouveau une description, collez dans URL de rappel l'URL ci-dessus et cliquez sur Save changes
  9. N'oubliez pas de renseigner votre nom d'utilisateur exact dans la configuration de Flickr2Piwigo
  10. Terminé!

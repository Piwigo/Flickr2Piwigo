1. Jděte na Flickr web a přihlašte se na váš účet
  2. Jděte na API Keys stránku
  3. Klikněte Get another key
  4. Zvolte Apply for a non-commercial key
  5. Vyplňte jmáno a popis, zaškrtněte obě zaškrtávací políčka a potvrďte
  6. Nyní zkopírujte Klíč a Tajnou frázi na konfigurační stránku Flickr2Piwigo
  7. Klikněte Edit auth flow for this app
  8. Zadejte znovu popis, do Callback URL zadejte URL adresu co je nahoře a klikněte Uložit změny
  9. Hotovo!

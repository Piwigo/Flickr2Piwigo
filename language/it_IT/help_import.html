Duplicati

Le fotografie che già esistono in Piwigo non verranno importate.

Il confronto viene effettuato prima sul nome del file (per una corrispondenza con "flickr-<Flickr_ID>") e poi sul checksum del reale file immagine. Si noti che nessun metadato viene aggiornato per le foto già importate.

Date approssimative

Flickr supporta le date approssimative per l'attributo "data di scatto" delle foto.

Verranno importate come date esatte e verrà aggiunta una parola chiave data approssimativa, oltre a parole chiave per la data (ad esempio c. 1933, marzo 2008 o 1986) e la granularità (una tra: mese, anno, circa).

Ad esempio, una data presa da Flickr di "c. 2006" verrà importata come data "2006-01-01 00:00:00" e le parole chiave "data approssimativa", "circa" e "c. 2006".

1. Choď na Flickr web stránku a pripojte sa na Vaše konto
  2. Choď na API stránku kľúčov
  3. Klikni Poslať iný kľúč
  4. Zvoľ Pridať pre nekomerčný kľúč
  5. Zadajte meno a popis a označte dva checkboxy pred odoslaním
  6. Skopírujte Key a Secret vložte do Flickr2Piwigo konfigurácie
  7. Klikni Upraviť auth flow pre túto aplikáciu
  8. Vložte popis opäť do URL spätné volanie URL a klikni Uložiť zmeny
  9. Hotovo!

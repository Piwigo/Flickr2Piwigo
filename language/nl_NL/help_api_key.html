1. Ga naar de Flickr website en log in.
  2. Ga naar de API keys pagina
  3. Klik op Get another key
  4. Kies Apply for a non-commercial key
  5. Geef hier een naam in en een beschrijving en vink de twee keuzevakjes aan alvorens op uitvoeren te klikken.
  6. Kopieer de sleutel (Key) en het geheim Secret naar de Flickr2Piwigo configuratie
  7. Klik op Edit auth flow for this app
  8. Geef opnieuw een beschrijving en paste bovenstaande URL in Callback URL en klik op Bewaar wijzingingen
  9. Klaar!

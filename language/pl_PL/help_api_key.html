1. Idź do Flickr website i zaloguj się
  2. Idź do API key pages
  3. Kliknij Get another key
  4. Wybierz Apply for a non-commercial key
  5. Wpisz nazwę, opis oraz zaznacz dwa checkboxy przed potwierdzeniem
  6. Teraz skopiuj Key i Secret do konfiguracji Flickr2Piwigo
  7. Kliknij Edit auth flow for this app
  8. Wpisz ponownie opis, wklej w Callback URL powyższy link i kliknij Save changes
  9. Gotowe!

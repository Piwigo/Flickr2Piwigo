1. Gå tilFlickrs hjemmeside og opret forbindelse til din konto
  2. Gå tilAPI-nøglesiden
  3. Klik på Get another key
  4. Vælg Apply for a non-commercial key
  5. Indtast et navn og en beskrivelse og kontroller de to afkrydsningsfelter før indsendelse
  6. Kopier Key og Secret til Flickr2Piwigo-opsætningen
  7. Klik på Edit auth flow for this app
  8. Indtast igen en beskrivelse, klistr ovennævnte URL ind i Callback URL og klik på Save changes
  9. Du er færdig!

1. Iet uz Flickr mājas lapu un pieteikties savā kontā
  2. Iet uz API atslēgu lapu
  3. Click Get another key
  4. Izvēlēties Pieteikties nekomerciālai atslēgai
  5. Ievadīt nosaukumu un aprakstu un pirms apstiprināšanas veikt atzīmes divās izvēles rūtiņās
  6. Tagad iekopējiet Atslēgu un Slepeno kodu Flickr2Piwigo konfigurācijā
  7. Klikšķiniet Rediģēt autorizācijas plūsmu šai aplikācijai
  8. Atkārtoti ievadīt aprakstu, agrākajam Callback URL augšējais URL un klikšķiniet Saglabāt izmaiņas
  9. Esat paveikuši!

1. Gehe auf Flickr und verbinde dich mit deinem Account
  2. Gehe auf die API-Keys-Seite
  3. Klicke auf Get another key
  4. Wähle Apply for a non-commercial key
  5. Gib einen Namen und eine Beschreibung ein und wähle die zwei Checkboxen vor dem Absenden aus
  6. Kopiere jetzt Schlüssel und the Geheimnis zur Flickr2Piwigo Konfiguration
  7. Klicke Edit auth flow for this app
  8. Gib erneut eine Beschreibung ein, füge die Callback URL ein und wähle Änderungen speichern
  9. Fertig!

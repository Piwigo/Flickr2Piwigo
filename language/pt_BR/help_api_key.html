1. Vá para página do Flickr e faça loguin na sua conta
  2. Vá para Pagina da chave API (API Keys)
  3. Clique em Get another key
  4. Escolha Aplicar uma chave não comercial (Apply for a non-commercial key)
  5. Entre com um nome e uma descrição e marque as duas caixas antes de enviar os dados.
  6. Agora copie a Chave (Key) e a chave secreta (Secret Key) para a configuração do Flickr2Piwigo
  7. Clique Edit auth flow for this app
  8. Entre com uma descrição novamente, cole na URL de Retorno (Callback URL) a URL acima e Salve as mudanças
  9. Você terminou!

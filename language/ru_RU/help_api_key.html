1. Откройте веб-сайт Flickr и подключитесь к своей учетной записи
  2. Перейдите на страницу ключей API
  3. Нажмите Get another key
  4. Выберите Apply for a non-commercial key
  5. Введите название и описание и отметьте два чекбокса галочками
  6. Скопируйте Key и Secret в настройку Flickr2Piwigo
  7. Выберите Edit auth flow for this app
  8. Введите описание снова, вставьте Обратный адрес URL и нажмите Save changes
  9. Все, готово!

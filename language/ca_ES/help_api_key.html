1. Aneu al lloc web de Flickr i connecteu-vos al vostre compte
  2. Aneu a la pàgina Apps By You (o Tus aplicaciones) a l'App Garden
  3. Feu clic a Get an API key (o Obtener una clave API)
  4. Seleccioneu Apply for a non-commercial key (o Solicitar una clave no comercial)
  5. Especifiqueu un nom i una descripció i marqueu les dues caselles de selecció abans d'enviar
  6. Copieu la Clau i el Secret als camps d'aquí dalt, i feu clic a Desa els ajustos
  7. Ara podeu anar a la pestanya d'importació, i se us demanarà que us connecteu a Flickr des de Piwigo

1. Πηγαίνετε στην ιστοσελίδα του Flickr και για να συνδεθείτε στο λογαριασμό σας
  2. Πηγαίνετε στη σελίδα API κλειδιά
  3. Κάντε κλικ στο Λήψη Ένα άλλο βασικό Λήψη άλλου κλειδιού
  4. Επιλέξτε Αίτηση για μη εμπορικό κλειδί
  5. Εισάγετε ένα όνομα και μια περιγραφή και τσεκάρετε τα δύο τετραγωνίδια πριν υποβάλετε
  6. Τώρα αντιγράψτε το κλειδί και το Μυστικό για τη διαμόρφωση του Flickr2Piwigo
  7. Κάντε κλικ στο κουμπί Επεξεργασία ροής ταυτοποίησης για αυτή την εφαρμογή
  8. Εισάγετε μια περιγραφή πάλι, επικολλήστε το URL επανάκλησης Το παραπάνω URL και κάντε κλικ στο Αποθηκεύστε τις αλλαγές
  9. Τελειώσατε!

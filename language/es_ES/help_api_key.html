1. Vaya al sitio web de Flickr y conéctese a su cuenta
  2. Vaya a la pagina "API keys"
  3. Haga clic en Obtener otra clave
  4. Seleccione Solicite una clave no comercial
  5. Escriba un nombre y una descripción y compruebe las dos casillas de verificación antes de enviar
  6. Ahora copie la clave y la contraseña en la configuración de Flickr2Piwigo/li>
  7. Haga clic en Editar el flujo de autenticación para esta aplicación
  8. Introduzca una nueva descipción, pegue la dirección de arriba en Callback URL y haga clic en Guardar cambios
  9. ¡Ya está!
